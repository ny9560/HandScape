# HandScape 🪄

AI Gesture-Controlled 3D Interface using **MediaPipe** and **Three.js**

HandScape is a real-time **gesture-controlled 3D interaction system** that allows users to manipulate virtual objects using hand gestures captured from a webcam.
The project uses **MediaPipe Hands** for hand tracking and **Three.js** to render interactive 3D shapes directly in the browser.

Users can interact with objects **without touching the screen** by using simple gestures like pinch, fist, and open hand.

---

## ✨ Features

* Real-time **hand tracking using MediaPipe**
* **Gesture recognition** for controlling objects
* Interactive **3D rendering with Three.js**
* **Pinch gesture** to resize objects
* **Right fist** to change object color
* **Left fist** to cycle between shapes
* Smooth **webcam-based interaction**
* Fully **browser-based system**

## 🚀 How to Run

1. Clone the repository

```
git clone https://github.com/ny9560/HandScape.git
```

2. Open the project folder

```
cd HandScape
```

3. Open **index.html** in your browser.

4. Allow **webcam access** and start interacting with gestures.

---

## 🎮 Gesture Controls

| Gesture         | Action              |
| --------------- | ------------------- |
| Right Hand Open | Resize object       |
| Right Fist      | Change random color |
| Left Fist       | Switch shapes       |
